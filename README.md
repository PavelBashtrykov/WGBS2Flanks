# WGBS2Flanks

This repository contains an exemplary pipeline and scripts to conduct flanking sequence analysis of methylome.

Methylation of genomic DNA is analysed using whole-genome bisulfite sequencing (WGBS). Obtained NGS data can be processed by Bismark (https://github.com/FelixKrueger/Bismark) or BMap (http://itolab.med.kyushu-u.ac.jp/DT/BMap/) programs to compute methylation levels of individual cytosines. Further, various downstream analyses can be carried out, for example, the identification of differentially methylated regions (DMRs) and the investigation of flanking sequences of methylated CpG sites (for more details, please read a review: https://pubmed.ncbi.nlm.nih.gov/34375615/).  

## Installation  

* Pipeline requires installation of external tools listed in `requirements.txt` file. Make sure that they are properly installed acoording to developer's instructions and their locations added to `PATH`.
* Pipeline is implemented in `wgbs_pipeline.py` script, which gets SRA ids (to download WGBS data from NCBI) and commands from `config.json`.
* Additional scripts `wgbs2bed.py`, `groom_bed.py`, `computeFlanks.py`, and `analyseFlanks.py` available here are needed to perform the analysis.

## wgbs2bed.py

Generates a bed file with coordinates of cytosine flanks from WGBS report files.

```
python wgbs2bed.py -i sorted.deduplicated.CX_report.txt -o coordinates.bed
```  

Options:
* `--infile / -i <str>`  
Name of an input file.  

* `--bmap`  
If given, then the script expects an infile generated by `BMap`. By default: `bismark` format is expected.  

* `--outfile / -o <str>`  
Name of an output file.  

## groom_bed.py

Takes the txt file produced by `bedtools getfasta` and formats it to make a `BED6` file.

```
python groom_bed.py -i sequences.txt -o sequences.bed6.bed
```  

Options:
* `--infile / -i <str>`  
Name of an input file.  

## computeFlanks.py

Computes frequences of cytosine methylation in different flanking contexts.

This script takes the `BED6` file and calculates methylation of cytosines for NCGN, NNCGNN and NNNCGNNN flanks. Firstly, a methylation level of idividual cytosines, which has sequencing depth >= a value defined by --depth parameter (default is 10), is computed. Secondly, average methylation of cytosines with identical flanks is calculated.
(Alternatively, takes as an input the txt file produced by `bedtools getfasta`, whithout grooming, but this has to be manuly changed in the `def main()`, this is an old version)

```
python computeFlanks.py -i sequences.bed6.bed -d 20 -o flanks.csv
```  
Options:  

* `--infile / -i <str>`  
Name of an input file.  

* `--outfile / -o <str>`  
Name of an output file.  

* `--depth / -d <int>`  
Sequencing depth threshold for CpG sites to be included in the analysis (default: 10).  

* `--motif / -m <str>`  
Motif to analyse, default is CG, other options are CA, CT, CC.  


## analyseFlanks.py

Compares flanking preference with reference data

This script takes the output from `computeFlanks.py` and csv file containing reference flanking data as comparison; generates a heatmap, a scatterplot, and a barplot.

```
python analyseFlanks.py -i flanks.csv -c preferences.csv
```  
Options:  

* `--infile / -i <str>`  
Name of an input csv file(s) generated by `computeFlanks.py`.  

* `--comparison / -c <str>`  
Name of the csv file containing flanks data to be used as comparison.
